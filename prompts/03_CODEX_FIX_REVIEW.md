# Codex prompt — Validate and fix review findings

Claude Codeのレビュー結果を受け取りました。

1. 各指摘を実際のコードと照合してください。
2. 妥当な指摘だけを修正してください。
3. 妥当でない指摘は、コード上の根拠を示して修正しない理由を説明してください。
4. 修正範囲をレビュー指摘に限定してください。
5. lint、typecheck、testがあればtest、buildを実行してください。
6. 回帰がないか主要4画面を確認してください。

完了時に以下を報告してください。

- 修正した指摘
- 修正しなかった指摘と理由
- 変更ファイル
- 検証結果
- 残るリスク
