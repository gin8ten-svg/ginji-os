# Product Requirements

## 1. Product name

仮称: Ginji OS

## 2. Problem

ToDoリストには「何をするか」は存在するが、実際に「いつ、何分やるか」まで毎日配置する負担が大きい。
固定予定や締切の変更により、作成した計画もすぐに崩れる。

## 3. Target user

初期段階では単一ユーザー向け。
大学、就職活動、仕事、事業、運動、私生活の予定を同時に管理する人。

## 4. Core value

- ToDoを時間枠へ変換する
- 固定予定を避けて現実的な一日を作る
- 予定が崩れたら再計画する
- 計画時間と実績時間の差から見積もり精度を改善する

## 5. MVP scope

### Included

- Googleログイン
- ToDo CRUD
- 期限、優先度、予想時間、カテゴリー
- Googleカレンダー読み取り
- 今日の空き時間表示
- 一日の計画案生成
- 計画案の手動編集
- 承認後のカレンダー書き込み
- 完了、延期、実績時間
- 日次振り返り

### Excluded

- チーム利用
- SNS管理
- メール自動処理
- 家計簿
- 複数AIエージェント会議
- AIによる固定予定の無断変更
- 有料プラン
- 一般公開
- ネイティブアプリ

## 6. Main screens

### Today

- 現在時刻
- 次の予定
- 今日のタイムライン
- 完了率
- 今日を組む
- 今から組み直す

### Tasks

- Inbox
- Today
- Upcoming
- Overdue
- Completed
- タスク作成・編集

### Calendar

- 固定予定
- AI作業枠
- 空き時間
- 日・週表示

### Planner

- 計画条件
- 生成された案
- 変更理由
- 承認・再生成

### Review

- 計画時間
- 実績時間
- 完了率
- 持ち越し
- 見積もり誤差

## 7. Non-functional requirements

- スマートフォン優先
- 主要画面は3タップ以内で到達
- AI生成結果は構造化データとして検証
- 既存カレンダー予定を無断変更しない
- 書き込み操作は履歴を残す
- ユーザー単位でデータを完全分離する
